# modern_icons
A set of icons for various services and applications, based on Simple Icons and Wikimedia resources

This is something I've made for another project I'm working on. I needed fancy icons with some modern look but I need them not to look too minimalistic. You are free to download and use them if you find them useful.

`app-icons` contains the icons in `.png` format, `512x512` 24bpp. `svg` is self-explaintory. I've started with 42 icons as of 2026-05-31 (May 31st), but will add more if I need to.
